# nazi-rise-causal-chain
Interactive causal network visualization of Nazi seizure of power: 416 events, 69 actors, 581 causal edges (1918–1934)

纳粹上台是二十世纪最具破坏性的政治事件之一，但其因果解释长期受困于两种叙事陷阱："希特勒中心论"将复杂历史压缩为个人传记；"结构决定论"将经济崩溃、凡尔赛和约等宏观条件直接等同于政治结果，忽视了中介机制与行动者的策略互动。

本项目以**网络分析方法**重构这段历史，提取 **416 个历史事件**、**69 位关键人物**、**581 条因果关联**，构建了一个有向异质网络，支持从宏观拓扑、时序演变、微观路径三个维度自由探索。

通过可视化分析，本项目揭示了纳粹夺权并非线性因果链，而是一个**多中心、非线性的因果网络**：

1. **"背后一刀"神话作为网络枢纽**（E012）：连接战败创伤、自由军团暴力、政治暗杀、纳粹意识形态的超级节点，辐射贯穿整个网络

2. **啤酒馆政变（1923）的双重效应**：短期失败导致党被禁，长期却通过审判宣传、狱中理论化、策略转向，反而强化了网络连通性

3. **1930年9月选举作为相变临界点**（E213）：纳粹从12席跃升至107席，同时激活企业界资助、军方试探、街头暴力、国会失序四条因果链，使分散网络突然耦合

4. **保守派精英的引狼入室悖论**：巴本、施莱谢尔、兴登堡等人物主动将希特勒纳入权力网络，每次策略操作都以为自己能控制流向，实则加速了纳粹对权力中枢的渗透
